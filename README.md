# mrkdwn2markdown

A library to convert Slack's block kit blocks to Markdown. Not perfect, but it'll be enough for my use cases.

## Reference

- [Formatting text for app surfaces | Slack](https://api.slack.com/reference/surfaces/formatting)

## Privacy
